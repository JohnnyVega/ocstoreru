// - Установка

1. Заливаем файлы к корень магазина
2. Редактируем файл: /export/index.php  
- В строке: require_once('../admin/config.php')  прописываем правильный путь до конфига админки.  
3. Даем права пользователю на управление модулем (делается через админку)
4. Включаем модуль: Админка -> дополнения -> импорт/экспорт -> Обмен данными с 1C v8.x (Управление Торговлей)

Модуль готов
